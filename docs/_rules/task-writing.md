# TASK WRITING RULES

When given a feature request, you MUST generate tasks.

## Rules
- Break feature into smallest executable tasks
- Each task must be completable within 1 hour
- Each task must have clear input/output

## Required Structure

# TASK: <name>

## Objective
What to achieve

## Target Files
Exact file paths

## Input
Request format

## Output
Response format

## Steps
1. Step-by-step actions (NO abstraction)

## Definition of Done
- Measurable conditions ONLY

## Constraints
- Must follow global constraints
