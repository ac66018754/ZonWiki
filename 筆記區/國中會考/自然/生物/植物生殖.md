# 被子植物（開花植物）生殖器官

## 1. 雄蕊 (Stamen) — 負責生產

### 花藥 (Anther)
- 位置：雄蕊頂端膨大的部分
- 功能：負責製造並儲存花粉粒（花粉粒裡面含有精細胞）

### 花絲 (Filament)
- 位置：支持花藥的長柄
- 功能：將花藥撐高，有利於風或昆蟲接觸花粉，幫助傳粉

## 2. 雌蕊 (Pistil) — 負責接收與發育

### 柱頭 (Stigma)
- 位置：雌蕊的最頂端
- 功能：接收花粉的平台，通常會有黏液或絨毛

### 子房 (Ovary)
- 位置：雌蕊基部膨大的部分
- 功能：內含胚珠（胚珠內有卵細胞），受精後子房發育成果實，胚珠發育成種子

## 受精後的變化

| 構造 | 受精後變化 |
|------|----------|
| 花粉精細胞 + 胚珠卵細胞 | 結合為受精卵，發育成胚 |
| 胚珠 (Ovule) | 發育為種子 |
| 子房 (Ovary) | 發育為果實 |
