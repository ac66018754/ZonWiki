# VS Code 常用快捷鍵

---

## C#（C# Dev Kit / OmniSharp）

| 快捷鍵 | 功能 |
|--------|------|
| `F12` | 跳到定義 |
| `Ctrl + F12` | 跳到實作 |
| `Shift + F12` | 查看所有參考 |
| `Alt + F12` | 預覽定義（Peek） |
| `F2` | 重新命名符號（跨檔案） |
| `Ctrl + .` | 快速修復 / 重構建議（加 using、產生方法等） |
| `Ctrl + Shift + Space` | 顯示參數提示 |
| `Ctrl + Space` | 觸發 IntelliSense |
| `F5` | 開始偵錯 |
| `Ctrl + F5` | 不偵錯直接執行 |
| `F9` | 切換中斷點 |
| `F10` | 逐步執行（Step Over） |
| `F11` | 逐步進入（Step Into） |
| `Shift + F11` | 跳出（Step Out） |
| `Ctrl + Shift + B` | 執行建置工作 |
| `Ctrl + Shift + P` → `Go to Symbol in Workspace` | 搜尋整個方案中的類別/方法 |
| `Ctrl + T` | 快速搜尋符號（類別、方法、屬性） |

## Markdown（Markdown All in One / 內建）

| 快捷鍵 | 功能 |
|--------|------|
| `Ctrl + B` | **粗體**（選取文字後） |
| `Ctrl + I` | *斜體*（選取文字後） |
| `Ctrl + Shift + V` | 開啟 Markdown 預覽 |
| `Ctrl + K, V` | 側邊開啟 Markdown 預覽 |
| `Ctrl + Shift + ]` | 提升標題層級（如 `##` → `#`） |
| `Ctrl + Shift + [` | 降低標題層級（如 `#` → `##`） |
| `Alt + C` | 切換勾選框 `[ ]` ↔ `[x]` |
| `Tab` | 清單縮排 |
| `Shift + Tab` | 清單取消縮排 |
| `Enter` | 自動接續清單項目 |
| `Ctrl + Shift + P` → `Create Table of Contents` | 產生目錄 |
| `Ctrl + Shift + P` → `Format Document` | 格式化 Markdown |

## VS Code 內建常用

### 一般操作

| 快捷鍵 | 功能 |
|--------|------|
| `Ctrl + Shift + P` | 開啟命令面板 |
| `Ctrl + P` | 快速開啟檔案 |
| `Ctrl + ,` | 開啟設定 |
| `Ctrl + `` ` | 開啟/關閉終端機 |
| `Ctrl + B` | 開啟/關閉側邊欄 |
| `Ctrl + Shift + E` | 切換到檔案總管 |
| `Ctrl + Shift + F` | 全域搜尋 |
| `Ctrl + Shift + G` | 切換到 Git 面板 |
| `Ctrl + Shift + X` | 切換到擴充功能 |

## 編輯

| 快捷鍵 | 功能 |
|--------|------|
| `Ctrl + X` | 剪下整行（未選取時） |
| `Ctrl + C` | 複製整行（未選取時） |
| `Ctrl + Shift + K` | 刪除整行 |
| `Alt + ↑ / ↓` | 移動整行上/下 |
| `Shift + Alt + ↑ / ↓` | 複製整行上/下 |
| `Ctrl + Enter` | 在下方插入空行 |
| `Ctrl + Shift + Enter` | 在上方插入空行 |
| `Ctrl + D` | 選取下一個相同的文字 |
| `Ctrl + Shift + L` | 選取所有相同的文字 |
| `Ctrl + /` | 切換行註解 |
| `Shift + Alt + A` | 切換區塊註解 |
| `Ctrl + Z` | 復原 |
| `Ctrl + Shift + Z` | 重做 |
| `Tab` | 增加縮排 |
| `Shift + Tab` | 減少縮排 |

## 多游標

| 快捷鍵 | 功能 |
|--------|------|
| `Alt + 點擊` | 新增游標 |
| `Ctrl + Alt + ↑ / ↓` | 在上/下方新增游標 |
| `Ctrl + D` | 逐一選取相同文字並新增游標 |
| `Ctrl + U` | 取消最後一個游標 |

## 搜尋與取代

| 快捷鍵 | 功能 |
|--------|------|
| `Ctrl + F` | 搜尋 |
| `Ctrl + H` | 取代 |
| `F3 / Shift + F3` | 下一個/上一個搜尋結果 |
| `Ctrl + Shift + H` | 全域取代 |

## 導覽

| 快捷鍵 | 功能 |
|--------|------|
| `Ctrl + G` | 跳到指定行數 |
| `Ctrl + Shift + O` | 跳到檔案中的符號 |
| `F12` | 跳到定義 |
| `Alt + F12` | 預覽定義 |
| `Shift + F12` | 查看所有參考 |
| `Ctrl + Tab` | 切換已開啟的檔案 |
| `Alt + ← / →` | 上一個/下一個編輯位置 |
| `Ctrl + Shift + \` | 跳到對應的括號 |

## 分割視窗

| 快捷鍵 | 功能 |
|--------|------|
| `Ctrl + \` | 分割編輯器 |
| `Ctrl + 1 / 2 / 3` | 切換到第 1/2/3 個編輯器群組 |

## 程式碼輔助

| 快捷鍵 | 功能 |
|--------|------|
| `Ctrl + Space` | 觸發自動完成 |
| `Ctrl + Shift + Space` | 顯示參數提示 |
| `F2` | 重新命名符號 |
| `Ctrl + .` | 快速修復 / 程式碼動作 |
| `Shift + Alt + F` | 格式化整份文件 |
| `Ctrl + K, Ctrl + F` | 格式化選取範圍 |

## 摺疊

| 快捷鍵 | 功能 |
|--------|------|
| `Ctrl + Shift + [` | 摺疊區塊 |
| `Ctrl + Shift + ]` | 展開區塊 |
| `Ctrl + K, Ctrl + 0` | 全部摺疊 |
| `Ctrl + K, Ctrl + J` | 全部展開 |
